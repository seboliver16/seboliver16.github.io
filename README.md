# seboliver.dev

Personal portfolio site for Sebastian Oliver — software engineer, serial founder, AI builder.

**Live:** [seboliver.dev](https://seboliver.dev)

## Stack

- Vanilla HTML/CSS/JS — no frameworks, no build step
- Google Fonts: JetBrains Mono + Newsreader
- Hosted on GitHub Pages

## Features

- **Command palette** (Cmd+K) — search and navigate sections, links, ventures
- **Expandable cards** — ventures and experience entries expand on click
- **Animated sea background** — flowing s13r characters in the margins
- **Responsive** — single column on mobile, two column on desktop
- **Live clock** — footer timestamp updates in real time

## Sections

| Section | Content |
|---------|---------|
| Bio | Background, current status |
| Ventures | Vahalla, Julie AI, RoomMe |
| Experience | Ramp, Amazon Bedrock, Deloitte, UVA |
| Content | YouTube, TikTok, Instagram |
| Connect | Email, LinkedIn, X, GitHub |
